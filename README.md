# News App 
Android mobile application that present different news from any country (us,egypt ,...)and can share this news and show more details about news and 
can search on any news and can choose filter for result of search (popularity,newest ,...).

# Tools & Libraries
- Android studio.
- Kotlin.
- MVVM.
- Retrofit.
- REST APIS.
- ViewBinding.
- PreferenceManager.
- Coroutine.
